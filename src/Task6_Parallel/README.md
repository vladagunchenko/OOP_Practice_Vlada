### Завдання 6 – Паралельне виконання

Мета:
Реалізувати паралельну обробку елементів колекції з використанням шаблону Worker Thread.

Реалізовано:

- Пошук максимального значення
- Пошук мінімального значення
- Обчислення середнього значення
- Використання черги задач (CommandQueue)
- Використання шаблону Worker Thread
- JUnit тестування
